## Penjelasan Projek:
- Soal 1 : Program Parkir Mobil
  - Merupakan program untuk mendata mobil yang parkir di sebuah mall
  - Fitur : Menambah data mobil yang akan parkir, Menampilkan mobil yang sedang parkir, Menghitung tarif parkir
- Soal 2 : Program Tebak Kata
  - Dapat memilih skala kesulitan dengan point (Easy = ×1, medium = ×2, dan Easy ×3) dan nyawa yang berbeda (Easy = 8, Medium = 6, Hard 4)
  - Sistematika game yaitu user memasukkan satu persatu sampai kata tersebut terselesaikan
  - User dapat meminta hint dengan mengetik "?" (Terdapat konsekuensi yakni -1 nyawa)
  - Terdapat leaderboard yang berisikan nama dan score
- Soal 3 : Sistem Manajemen Inventaris Toko
  - Fitur : Tambah barang, menampilkan barang, mencari barang termahal, menghitung total nilai inventaris, cek jumlah barang untuk di restock, update stok barang.
- Soal 4 : Minesweeper
  - Memulai permainan baru, bisa mengatur ukuran grid dan jumlah bom
  - Membuka dan menandai cell
  - Menampilkan informasi lama permainan
- Soal 5 : Tebak Negara
  - Terdapat clue huruf negara yang akan ditebak, tiap negara terdapat 3 clue huruf yang berbeda-beda
  - Tiap jawaban benar dapat 1 poin, salah tidak dapat poin dan output jawaban yang benar
  - Terdapat konfirmasi apakah user ingin mengulang atau tidak, apabila tidak maka program output poin total
